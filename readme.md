# 20 Days Challenge

## Overview
The **20 Days Challenge** repository contains a collection of practice projects and proof-of-concept (POC) files completed as part of a 20-day learning challenge. The purpose of this repository is to improve technical skills through consistent daily practice and hands-on implementation.

## Objective
This challenge was created to:

- Build coding consistency through daily practice
- Improve problem-solving and project development skills
- Gain practical experience by working on multiple mini projects
- Strengthen understanding of different technical concepts
- Create a portfolio of learning-based projects
